If validity is dependent (upon something else), there can be no infinite regress. Hence there can be certainty regarding self-validity only when there is cognition of effective action.тАФ(2957)

тАЬWhat peculiar feature is there in the second cognition, different from those of the previous cognition, on account of which it is not regarded to be entirely like the previous cognition?тАЭтАФ(2958)
